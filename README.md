# Eco-Leaf 🌱

**A Climate Action & Conservation NGO Website**

Eco-Leaf is a Kenyan-based non-governmental organization dedicated to environmental conservation, climate action, biodiversity protection, and youth empowerment.

---

## 📌 About the Project

This is the official website for **Eco-Leaf**, founded in 2025 by Edgar Kamau. The website showcases the organization's mission, flagship projects, services, impact, and ways for people to get involved.

### Key Features
- Clean, modern, and responsive design
- Mobile-friendly interface
- Sections for Projects, Services, Impact, Get Involved, and Contact
- Focus on climate action, reforestation, and youth engagement in Kenya

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Responsive Design (Mobile-First)
